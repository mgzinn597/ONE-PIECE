# ONE-PIECE Esse projeto é uma página interativa dedicada a *One Piece*, focada na apresentação dos principais membros dos Piratas do Chapéu de Palha. Ele exibe uma seção com cinco personagens (Chopper, Zoro, Luffy, Nami e Sanji), cada um com uma imagem, nome e descrição.  

A interface permite ao usuário alternar entre os personagens por meio de botões com imagens representando cada tripulante. O personagem selecionado recebe destaque na tela. O código inclui:  

- **HTML** estruturado para exibir os personagens e botões de seleção.  
- **CSS** para estilizar a página, incluindo um layout responsivo.  
- **JavaScript** (não fornecido, mas referenciado) para tornar a troca de personagens interativa.  

O objetivo principal do projeto é criar uma experiência visual e dinâmica para fãs de *One Piece*, permitindo explorar informações sobre os personagens de forma interativa.
